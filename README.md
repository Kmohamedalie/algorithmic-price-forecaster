# Quantitative Stock Prediction Tools

This project is an interactive web application built with Streamlit that allows users to forecast stock prices using three distinct mathematical approaches:

- Statistical Analysis (ARIMA)

- Machine Learning (Random Forest Regressor)

- Additive Time-Series Modeling (Facebook Prophet)

Users can input any valid stock ticker, fetch live market data via yfinance, and visually compare how each algorithm handles short-term price predictions.

⚠️ Disclaimer: This application is built strictly for educational and data science demonstration purposes. Financial markets are highly volatile, and no algorithm can predict stock prices with 100% certainty. Do not use these models for real-world financial trading.
